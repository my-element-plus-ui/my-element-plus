RCTable 表格组件 ​

由操作按钮组、基础表格、列求和、列筛选、列复制、列操作等集合多功能的表格。实现表头吸顶，滚动条和翻页吸底，方便用户操作。

tableList 配置规则 ​

  • key: 表格列对应的key
  • label: 表格列对应的label
  • isShow: 表格列是否显示
  • width表格列宽度
  • align表格列对齐，可选left right center
  • type: 表格列类型
    • indexNum: 序号，默认1开始排序，自动递增。
    • link: 链接列，通过callback回调函数做对应操作或跳转。
    • textAndInput: 集合text和inputNumber, key对应的是inputNumber双向绑定的值，textKey对应的是text的值，当inputNumber改变时通过changeInput回调做对应操作。
    • inputNumber: 数字输入框，通过changeInput回调函数做对应操作。
    • select: 下拉框选择框，selectKey为staticData的静态枚举的key值，通过onChangeSelected回调函数做对应操作。
    • date: 日期自动转换为默认YYYY-MM-DD HH:mm:ss格式，可通过format设置日期转换格式，例YYYY-MM-DD。
    • enumSelect: staticData静态枚举默认取key的枚举，如果设置了selectKey,则取selectKey的枚举， 设置showAll为true则没有匹配到的枚举也显示，否则不显示；设置showBoth为true则以【value-name】的格式显示，否则只显示name,staticData的枚举项设置type，可显示icon颜色小方块，可选info、danger、success、primary、warning。
    • colorAndText: 显示icon颜色小方块和text，setColorType设置颜色小方块函数，return值可选info、danger、success、primary、warning。
    • colorText: 显示不同颜色的字体text，setColor设置字体颜色函数，return颜色css。
    • process: 显示百分比进度条，setColor设置进度条颜色函数，return颜色css。
    • checkBox: 用于设置true false值，只用于boolean,不可编辑。
  • hoverMore: object类型，配置列筛选isFilter,列求和isSum,列排序isSort,列属性isCopy
    • isCopy: 列复制，boolean类型，通过设置true, false，配置是否有列复制功能，默认false。
    • isSum: 列求和，object类型，通过配置RCTable Attributes 的sumUrl作为求和接口返回，亦可配置isSum的sumUrl作为单独列求和接口。sumKey为接口取值的key，subSumKey为接口取值的第二个key,key/subSumKey的格式显示。
    • isSort： 列排序，object类型，sortKey为筛选字段，sortType为辅助字段。传参后端格式为{ "sortArrParams": [{ "key": "goods_num", "sort": "desc", "type": "ext" }]}
    • isFilter: 列筛选，object类型，配置项如下。
      • filterKey： 筛选key
      • type: 列筛选弹窗表单类型。
        • radio: 单选，staticData静态枚举默认取staticType设置key的枚举
        • checkbox: 多选，staticData静态枚举默认取staticType设置key的枚举
        • basicSelect: 弹窗选择框，ruleOutKey为排除项key，selectKey为选中key，ruleOutSelectKey为排除项选中key，showLabel为显示在input的key，searchData为弹窗配置项，参考组件配置项BasicSelectModal 弹窗选择器。
        • searchSelect: 模糊下拉弹窗选择框，ruleOutKey为排除项key，selectKey为选中key，ruleOutSelectKey为排除项选中key，showLabel为显示在input的key，其他配置项参考组件配置项SearchSelect 模糊下拉弹窗选择器。
        • dateSelect: 日期选择器，filterKey可设置用&分割前后时间范围对应的key，若没有分割字符&，则以数组格式返回。format为时间格式化规则。
        • input: 输入框
        • digitalRange: 数字区间输入值，filterKey可设置用&分割前后数字范围对应的key
        • digitalRangeColor: 在digitalRange基础上添加对应的快速选中颜色方块。staticData静态枚举默认取staticType设置key的枚举,枚举项例子goodsNum: [ { min: 0, max: 0, type: "danger" }, { min: 1, max: 99, type: "warning" }, { min: 100, max: 100, type: ]
  • children: 配置多表头下的子表格。多表头近支持值回显，需要通过renderFunc处理返回值。

buttonList 配置规则 ​

  • key: 按钮唯一的key
  • label: 按钮名称
  • disabledFunc: 禁用函数,入参row为行数据，item为按钮配置，返回true为按钮禁用，false为按钮启用
  • showGreyCss: 禁用情况下按钮是否置灰，默认不显示。
  • callback: 点击按钮回调。

headButtonList 配置规则 ​

  • key: 按钮唯一的key
  • label: 按钮名称
  • needSelected: 选中按钮才启用
  • buttonProps: 按钮的配置属性。参考组件 Button 按钮 和 MenuButton 下拉菜单按钮
  • selectRule: 按钮是否禁用规则，例如selectRule: { status: ['状态1', '状态2'] } 当选中status 为 状态1或状态2 按钮才启用，选中的行只有一行不符合，按钮都禁用。
  • headButtonDisabledFunc: 按钮禁用函数，true为按钮禁用，false为按钮启用
  • type: 按钮类型,默认为Button 按钮，可选menuButton
    • menuButton，通过配置buttonProps来配置下拉按钮，menuList配置参考MenuButton 下拉菜单按钮，另外添加了needSelected是否选中数据按钮才启用。
  • callback: 点击按钮回调
